# Overview📃
This project demonstrates an automated CI/CD pipeline using GitHub Actions. 
The pipeline automatically builds, tests, and runs automation scripts whenever code is pushed to the repository. 
It helps improve deployment reliability and development efficiency.

# Technologies Used🛠️
->GitHub

->GitHub Actions

->Python

->Flask

->Linux

->YAML

# CI/CD Workflow🤖
The pipeline runs automatically on every push to the main branch and performs the following steps:
1) Checkout repository
2) Setup Python environment
3) Install dependencies
4) Run unit tests
5) Execute automation script
6) Build and deploy application

#Run Locally💻
1) Clone the repository:
https://github.com/ANKITSHARAN134/DevOps-CI-CD-project.git
cd devops-ci-cd-project

2) Install dependencies:
pip install -r requirements.txt

3) Run the application:
python app/app.py
